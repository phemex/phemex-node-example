# phemex-node-example

phemex rest and websocket examples by node.js

# Get Starting

- Install dependencies

```shell script
npm install
```

- Set your apiKey and secret in src/config.js

- Update src/index.js, choose rest sample or websocket sample

- Run sample

```shell script
npm start
```
